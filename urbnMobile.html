About URBN Mobile

This project showcase the application of skills and knowledge acquired throughout my previous work. I focused on efficient design processes, including advanced use of Figma. URBN Mobile concept (developed with permission), were undertaken to further refine my mobile design workflow.

URBN Mobile

Building URBN Mobile: The Why Behind the What

The design for URBN Mobile builds upon our group's previous work on the URBN web version. However, to optimize for mobile use, I made specific adjustments. For example, the original lighter primary color was darkened to #6A0DAD, enhancing readability and contrast on smaller screens. This adaptation, along with maintaining consistent fonts and assets, ensures a familiar user experience.

Behind the Interface: URBN Mobile's System Logic

The system logic for URBN Mobile mirrors the original web version. While the web version was fully developed, these mobile designs are conceptual. Page connectivity and core logic remain consistent, with minor additions and tweaks that maintain user familiarity. The primary mobile enhancement is the introduction of item/product reviews, a feature I decided to add as it is commonly found in most e-commerce mobile apps, and absent from the web version.
